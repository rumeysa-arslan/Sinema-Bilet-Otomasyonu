# 🎬 Sinema Bilet Satış Otomasyonu

Bu proje, bir sinema salonunun bilet satış, müşteri yönetimi ve film kayıt işlemlerini dijitalleştirmek amacıyla geliştirilmiş kapsamlı bir **C# Windows Forms** masaüstü uygulamasıdır. Kullanıcı dostu arayüzü ve ilişkisel veritabanı mimarisi ile sinema işletmeciliğini kolaylaştırmayı hedefler.

## ✨ Özellikler

* 🍿 **Dinamik Koltuk Seçimi:** 16 koltuklu salon düzeni. Seçilen koltuklar yeşil, daha önce satılmış (dolu) koltuklar kırmızı, boş koltuklar ise beyaz renkte gösterilir.
* 🎥 **Film İşlemleri:** Vizyondaki filmleri; ad, yönetmen, seans saati ve bilet fiyatı bilgileriyle sisteme ekleme.
* 👥 **Müşteri Yönetimi:** Müşterileri ad, soyad ve telefon numarası ile veritabanına kaydetme.
* 🌽 **Büfe Satışı:** Bilet satışı esnasında tek tıkla "Mısır (+50 TL)" ekleme ve anlık toplam tutar hesaplama.
* 📊 **Detaylı Satış Raporu:** `JOIN` sorguları kullanılarak hazırlanan; hangi müşterinin, hangi filmi, saat kaçta ve hangi koltukta izlediğini gösteren özel tasarımlı (bordo-siyah temalı) rapor ekranı.

## 🛠️ Kullanılan Teknolojiler

* **Programlama Dili:** C# (.NET Framework / .NET Core)
* **Arayüz (UI):** Windows Forms Application (WinForms)
* **Veritabanı:** Microsoft SQL Server (MSSQL)
* **Geliştirme Ortamı:** Visual Studio 2022
